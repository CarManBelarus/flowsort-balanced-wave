# flowsort-balanced-wave (функцыя сартавання трэкаў асабістых плэйлістоў у Spotify)

DJ-friendly tempo & key sorting tracks for Goofy / Spotify.

Гэты скрыпт рэалізуе функцыю `FlowSort.sortBalancedWave` — сартаванне трэкаў у асабістых плэйлістах з улікам тэмпу, танальнасцей (Camelot key) і базавых DJ-сцэнарыяў плыўнага развіцця пераходаў па танальнасцях.

(Функцыя моцна спрашчае расстаноўку трэкаў для плыўных, практычна бясшвоўных пераходаў паміж імі і дае магчымасць выкарыстання функцыі "стварыць мікс" у аўтаматычным рэжыме з добрым вынікам).

Падыходзіць для вялікіх плэйлістоў (да 4500 трэкаў) і выкарыстоўваецца ў экасістэме Goofy / Google Apps Script для працы са Spotify.

Для працы скрыпта патрабуецца прэміум-падпіска Spotify, настроены праект [Goofy](https://chimildic.github.io/goofy/#/install) з прыватнымі ключамі (да слова, велізарная падзяка аўтару праекта Goofy, без яго нічога гэтага не было б).

Змесціва файла `flowsort_sortBalancedWave.js` трэба скапіяваць, захаваць у асобны файл Apps Script і перасунуць яго вышэй у спісе, каб ён ішоў адразу пасля `library.gs`.

Выклікаць функцыю ў скрыптах трэба непасрэдна перад фарміраваннем (стварэннем) фінальнага плэйліста для зменнай, якая змяшчае трэкі фіналу (па прынцыпе `tracks = FlowSort.sortBalancedWave(tracks);`).

На скрыншоце ніжэй паказаны прыклад стварэння плэйліста з адсартаванымі любімымі трэкамі.

<img width="788" height="505" alt="Прыклад выкліку" src="https://github.com/user-attachments/assets/05a8956e-8b75-4917-b2eb-8b649b50ee73" />

**П.С.** Адсартаваны плэйліст рэкамендую слухаць з уключаным *crossfade = 10s* у наладах Playback Spotify, альбо выкарыстоўваць функцыю Spotify - Mix (у залежнасці ад разнастайнасці жанраў плэйліста. Калі плэйліст вельмі разнажанравы і трэкі моцна адрозніваюцца па BPM, то лепш выкарыстоўваць толькі crossfade).
